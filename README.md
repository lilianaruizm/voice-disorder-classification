# Voice Disorder Classification Using Deep Learning

## Overview

This project was developed as part of my Master's Thesis in Computer Science at Instituto Tecnológico de Hermosillo.

The objective was to develop an intelligent system capable of classifying healthy and pathological voice recordings using machine learning and deep learning techniques.

---

## Technologies

- Python
- MATLAB
- TensorFlow
- Scikit-Learn
- Librosa
- NumPy

---

## Methodology

1. Acoustic signal processing
2. Feature extraction from voice recordings
3. Data augmentation to address class imbalance
4. Development of MLP and CNN models
5. Performance evaluation using multiple classification metrics

---

## Features Extracted

- Fundamental Frequency (F0)
- Formants (F1-F3)
- Jitter
- Shimmer
- Harmonic-to-Noise Ratio (HNR)
- MFCC
- Delta MFCC
- Spectral Centroid
- Spectral Bandwidth

---

## Results

### CNN Model

- Test Accuracy: 96.3%
- ROC-AUC: 0.98

### Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix
- ROC-AUC
